# Pilot Plan — Sprint 0

**Scope:** 3 schools, ~150 students, 6 weeks
**Cards:** 40 Grade 1 cards (20 Kiswahili, 20 English)

## Success Metrics
| Metric | Target |
|--------|--------|
| First-attempt accuracy | 60–85% |
| Mastery within 3 attempts | ≥70% |
| Abandonment rate | ≤10% |
| Teacher satisfaction | ≥4/5 |
| Usable on 1GB RAM device | ✓ |

## Gates
- Flag cards with mastery <60% for content revision
- Flag cards with abandonment >10% for UX review

## Feedback cadence
- Week 2: teacher survey + telemetry review
- Week 4: mid-point KPI check
- Week 6: final report + card revision list
